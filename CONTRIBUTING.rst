Adding License Headers
----------------------

When creating new files, please include the following SPDX header at the top:

**For Dockerfiles, shell scripts, Python files, and YAML files:**

.. code-block:: bash

    # SPDX-License-Identifier: GPL-3.0-or-later
    # Copyright (C) <year> <your name> <your email>

How to Contribute
-----------------

1. Fork the repository
2. Create a new branch for your feature or bug fix
3. Make your changes, ensuring you add appropriate license headers
4. Test your changes
5. Submit a pull request

Questions?
----------

If you have any questions about contributing or licensing, please open an issue on the project repository.
